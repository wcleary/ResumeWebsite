The SocialBoard is a website I made in my first software methodologies class. Five classmates and I worked together to create a social media hub where you are able to access your photos and posts from Facebook, Twitter and Instagram. (Instagram has lost functionality due to changes in their api). As a developer on the team, I worked on the front and back-end configuring the api connection and authentication in PHP. I also worked on the front-end JavaScript and HTML creating user firendly functionality for displayed media such as photos,tweets, likes and comments.

LuaGuard

LuaGuard is a Lua obfuscator written in Java. LuaGuard was a group project with 6 classmates and 3 students from Peking University in Bejing, China. This project implemented multiple OO design patterns in order to keep the project maintainable and scalable. I was responsible for creating the dynamic Java GUI in swing and for creating multiple obfuscations.
